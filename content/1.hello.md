## nuxt content

nuxt content 为我们提供了一个基于文件的 CMS

### nuxt content 路由

nuxt content 通过映射 `/content/` 目录的文件名为路由，方便我们访问对应页面

### show me hello world code

万事开头 hello world

**代码**

```ts
const hello = (name = 'world') => {
  return `hello ${name}`
}

console.log(hello())
```

**结果**

```bash
hello world
```
